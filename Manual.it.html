01. Prima di iniziare

Cosa fa VoiceBridge

VoiceBridge trasforma documenti in audio, trascrive audio/video, crea sottotitoli e permette piccoli interventi manuali su audio e video.

L'app è organizzata per workflow. In genere si sceglie una sezione, si carica un file, si controllano le opzioni essenziali e si avvia il job.

Requisiti operativi

  • Edge TTS richiede internet.
  • Local TTS richiede XTTS-v2 locale.
  • Transcription richiede Whisper large-v3.
  • SRT richiede anche il modello di allineamento della lingua.
  • Audio/Video Cleanup usano ffmpeg.
  • Video glitch detection usa OpenCV.

Prerequisiti opzionali

ComponenteQuando serve
Microsoft WordSolo per vecchi documenti .doc.
Tesseract OCRSolo per PDF scansionati o basati su immagini.

La risoluzione minima pratica è Full HD, 1920x1080. Su schermi più piccoli o con scaling alto alcune schermate possono richiedere più attenzione.

02. Home e sidebar

La Home mostra lo stato operativo dell'app: Edge TTS, runtime locale, modelli STT, Local TTS, DVAE e strumenti video. Se qualcosa manca, la sezione interessata mostra il messaggio o il pulsante necessario.

Sezioni principali

  • Text to Speech: genera MP3 da documenti.
  • Local Voices: gestisce profili, dataset, setup e training.
  • Transcription: crea transcript Markdown/Word o sottotitoli SRT.
  • Subtitles: aggiunge o imprime SRT nei video.
  • Audio Cleanup: corregge piccoli difetti audio.
  • Video Cleanup: corregge singoli frame problematici.
La lingua dell'interfaccia si cambia dalla sidebar. I campi file e output partono vuoti a ogni avvio per ridurre il rischio di lavorare sul file sbagliato.

03. Text to Speech

  1. Apri Text to Speech.
  2. Seleziona un file .txt, .docx, .doc o .pdf.
  3. Scegli engine, voce, lingua e velocità.
  4. Scegli il percorso di salvataggio dell'MP3.
  5. Premi Generate MP3.
  6. A fine job usa Open output, Open folder o Open in Audio Cleanup.
Generate MP3 diventa disponibile solo quando il file input esiste e la voce scelta è pronta.

Edge TTS

Edge TTS usa le voci Microsoft online. Se internet non è disponibile, VoiceBridge disabilita Edge TTS, passa temporaneamente a Local TTS quando possibile e ritenta il controllo della lista voci in background.

Local TTS

Local TTS usa XTTS-v2 nel runtime locale. Prima dell'uso scarica il modello con Download XTTS-v2, poi scegli profilo vocale, preset e device.

PresetUso consigliato
StablePiù conservativo, utile per test lunghi.
BalancedCompromesso consigliato.
NaturalPiù espressivo, potenzialmente meno stabile.

Multi-voice blocks

Divide il testo in blocchi e permette di assegnare voce e velocità diverse. L'output resta un unico MP3 finale. Quando disponibile, viene salvata anche una timeline JSON accanto all'MP3.

04. Local Voices

Local Voices raccoglie Profiles, Datasets, Setup e Training.

Profiles

  1. Crea un profilo Reference clone se hai già un audio reference autorizzato.
  2. Crea un profilo Modeling dataset se vuoi raccogliere clip per un futuro training.
  3. Per registrare un reference audio, usa Record.
  4. Ascolta il WAV pulito e scegli se mantenerlo, ritentare o annullare.
Il tipo profilo si sceglie alla creazione e resta bloccato. Se elimini un profilo con lavoro di modeling collegato, l'app chiede conferma distruttiva prima di eliminare anche dataset e asset identificabili.

05. Local Voices > Datasets

  1. Crea prima un profilo Modeling dataset.
  2. Apri Local Voices > Datasets.
  3. Usa Generate guided text per creare un prompt locale adatto alla lingua.
  4. Usa Record from text per registrare la clip guidata.
  5. Ascolta il risultato, poi mantieni o ritenta.
  6. Correggi eventuali transcript e usa Verify text se serve.
  7. Usa Exclude export per tenere una clip nel dataset ma non esportarla.
Regola d'oro per la registrazione: leggi in modo naturale e chiaro, come un audiolibro rivolto a una sola persona. Parla leggermente più lentamente del parlato quotidiano, ma senza scandire artificialmente. Rispetta la punteggiatura con pause naturali: breve sulla virgola, più netta sul punto, lieve intonazione sulle domande. Evita enfasi teatrale, voce monotona, cambi di volume, cambi di distanza dal microfono e rumori di fondo.

Setup Dataset è disponibile solo quando il dataset raggiunge la readiness minima. Esporta le clip pronte quando il dataset è cambiato, poi apre Setup. Se l'export corrente è già aggiornato, il pulsante diventa Open Setup e non ricopia i file.
I prompt guidati generati dall'app sono limitati a 200 caratteri per restare compatibili con il fine-tuning XTTS-v2. Se incolli o carichi testo personalizzato, tieni ogni clip entro lo stesso limite: gli export con transcript più lunghi vengono bloccati in fase di preparazione training per evitare errori del trainer.

06. Setup e Training

Setup

  1. Apri il dataset dal suo profilo, poi usa Setup Dataset.
  2. Imposta output, device, epoch e batch size. L'app propone valori iniziali in base alla durata del dataset e alla VRAM CUDA rilevata.
  3. Scarica asset training se mancano.
  4. Esegui Refresh preflight.
  5. Salva con Save Config. Se non ci sono modifiche visibili alla configurazione, il pulsante resta disattivato.
  6. Usa Go to Training quando è disponibile una config salvata.

Training

  1. Apri Training da Setup con Go to Training.
  2. Usa Prepare.
  3. Usa Dry run per verificare senza addestrare.
  4. Usa Start training.
  5. Usa Cancel per terminare il processo esterno.
  6. Usa Open job folder per controllare log e output.
Il download degli asset training può essere annullato con Cancel download. Se CUDA fallisce, l'app può proporre di passare il job a CPU e ritentare.
I pulsanti seguono lo stato del job: dopo Save Config e Go to Training è disponibile solo Prepare; dopo Prepare è disponibile solo Dry run; dopo Dry run completato correttamente è disponibile Start training. Se il job fallisce o viene annullato, riparti da Prepare.
A training completato l'app mantiene il modello in inference_model, archivia i log in logs/voice_modeling e rimuove i checkpoint intermedi della cartella run/training per evitare accumuli da molti GB.

07. Transcription

  1. Apri Transcription.
  2. Se compare Download Whisper large-v3, scarica prima il modello.
  3. Seleziona audio o video.
  4. Scegli modalità: Markdown, Word, SRT automatico o SRT da testo fornito.
  5. Scegli lingua e device.
  6. Avvia il job.
ModalitàQuando usarla
Transcript Markdown (.md)Trascrizione leggibile, senza timing SRT.
Transcript Word (.docx)Transcript leggibile in formato Word.
Auto subtitles (.srt)Sottotitoli temporizzati automaticamente.
Subtitles from provided text (.srt)Hai già un testo e vuoi allinearlo ad audio/video.
Le modalità SRT richiedono il modello di allineamento della lingua parlata. Se manca, l'app chiede se scaricarlo.
Generate diventa disponibile solo quando la preflight STT è completata e i file richiesti dalla modalità scelta esistono.

08. Subtitles

  1. Seleziona video sorgente.
  2. Seleziona file .srt.
  3. Scegli output.
  4. Usa Embed SRT track per una traccia selezionabile.
  5. Usa Burn in SRT per imprimere i sottotitoli nel video.
  6. Nel burn-in regola posizione, dimensione, outline, ombra, colori e qualità.

Il burn-in ricodifica il video. Le modalità CRF più basse aumentano qualità e dimensione file.

Create video diventa disponibile solo dopo aver selezionato un video esistente e un file .srt valido.

09. Audio Cleanup

  1. Seleziona un file audio supportato.
  2. Usa waveform, zoom o campi Start/End per selezionare un range.
  3. Se esiste una timeline TTS, usa TTS blocks.
  4. Premi Cut, Silence o Fade.
  5. Ripeti per più correzioni.
  6. Controlla Applied changes.
  7. Premi Clean audio.
Le modifiche non possono sovrapporsi. Gli stage intermedi usano formato lossless; l'audio viene codificato nel formato finale solo all'ultimo passaggio.

10. Video Cleanup

  1. Seleziona il video.
  2. Scegli output e qualità.
  3. Usa Frame review per caricare la filmstrip.
  4. Usa le frecce vicino allo slider per muoverti di un frame alla volta.
  5. Usa Detect black frames per trovare frame neri isolati.
  6. Usa Detect frame glitches per trovare frame sospetti non neri.
  7. Fai doppio click su un risultato per saltare al frame.
  8. Marca manualmente i frame da correggere quando serve.
  9. Applica Freeze previous frame o Remove selected frames.
  10. Controlla Applied changes e premi Clean video.
Detect black frames auto-marca solo frame neri isolati considerati riparabili. I frame sospetti non neri sono solo segnalati e vanno verificati manualmente.
AzioneEffetto
Freeze previous frameSostituisce il frame marcato con quello precedente e mantiene la durata originale.
Remove selected framesRimuove frame e micro-porzioni audio corrispondenti, accorciando leggermente la timeline.

Risoluzione rapida

  • Edge TTS non disponibile: controlla internet oppure usa Local TTS.
  • Local TTS non disponibile: scarica XTTS-v2 e verifica il profilo vocale.
  • Transcription non pronta: scarica Whisper large-v3.
  • SRT non generato: verifica lingua e modello di allineamento.
  • Audio/video cleanup fallisce: controlla output, spazio disco e file aperti da altri programmi.
  • CUDA non disponibile: usa Auto o CPU.
  • PDF vuoto o scansionato: installa Tesseract OCR e usa OCR quando richiesto.
